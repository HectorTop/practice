KDE graphs - программа, которая строит графики KDE для dx,dy,x,y для обоих выборок
MNRAS - программа которая создает синтетическую выборку, подобную выборке MNRAS и показывает графики для оригинальной и синтетической выборки
PAZH - программа которая создает синтетическую выборку, подобную выборке PAZH и показывает графики для оригинальной и синтетической выборки
